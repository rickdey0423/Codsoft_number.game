# 🎮 Codsoft Number Guessing Game

A fun, interactive number guessing game built with HTML, CSS, and JavaScript. Featuring kid-friendly animations and real-time feedback!

## 🌐 Live Preview
**[👉 Click Here to Play the Game Live!](https://rickdey0423.github.io/Codsoft_number.game/)**

---

## 🚀 Features
- **Responsive Design:** Works on mobile and desktop.
- **Interactive:** Built-in animations using Animate.css.
- **User Friendly:** Press 'Enter' to submit your guess.
- **Smart Logic:** Tells you if your guess is too high or too low.

## 🛠️ Installation
If you want to run this locally:
1. Clone the repo: `git clone https://github.com/rickdey0423/Codsoft_number.game.git`
2. Open `index.html` in your browser.
